This web application is a centralized platform designed to streamline and manage the end-to-end interview process for multiple practices within an organization. It enables Practice Admins, Panel Members, and the TA (Talent Acquisition) team to coordinate effectively and ensures efficient scheduling, tracking, and feedback management.

Main Modules

Authentication & Authorization Login / Logout Role-based access (Practice Admin, Panel Member, TA Team) User Management (create/edit/delete users, assign roles)

Practice Management Create / Edit / Delete Practice - Adim User Assign Practice Admins - Admin User Map panel members to practices

Role & JD Managemen Create / Edit / Delete Designation (for hire) Create / Edit / Delete Job Descriptions (JDs) for Designation Associate JD with Designation and practice

Interview Panel Management Add / Edit / Delete panel members Map panel members to roles and interview levels Assign panel members to interviews View panel availability

Interview Slot Management Admin View: View slots by day/week/month/year (filters: role, level) Add/Edit slots on behalf of panel members Panel View: Add/Edit/Delete own slots Mark unavailable (for existing slots) View upcoming interviews Reschedule request TA Team View: View slots by day/week/month/year (filters: role, level) Block slots Release slots with reason Schedule Interview

Interview Scheduling

Create interview opportunities Map profiles to opportunities Block slots Schedule interview Cancel interview with reason Reschedule interviews

Feedback Management

Master Templates: Create / Edit Feedback templates per role

Panel View: Add feedback after interview View previous feedbacks for a profile

TA Team View: View / Download interview feedback

User Roles & Key Features

Role	Key Features
Practice Admin	Create/Edit roles, JDs, panel mapping, slots overview, feedback template creation, view feedbacks
Panel Member	Manage availability, view schedule, request reschedule, conduct interview, add feedback
TA Team	View/manage slots, create opportunities, map profiles, schedule/reschedule/cancel interviews, manage feedback
Master Admin (Optional)	System-wide settings, feedback templates, reporting, practice management
